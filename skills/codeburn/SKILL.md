# CodeBurn

Use this skill when the user asks about token cost, context bloat, or workflow
efficiency.

## Daily Discipline

- Keep prompts scoped to one task.
- Do not paste full files when paths and focused excerpts are enough.
- Summarize long logs before handing them to another agent.
- Prefer repository instructions over repeated prompt boilerplate.

## Review Cadence

Review token and cost usage weekly or after unusually expensive sessions.

Look for:

- Repeated setup prompts that should become repo instructions.
- Long histories that should be summarized.
- Agents re-reading the same files.
- Overuse of powerful models for cheap inspection tasks.

## Tool Use

If CodeBurn or a similar dashboard is installed, use it to inspect sessions and
projects. If absent, summarize cost drivers from logs, transcripts, or command
history.
