# CodeBurn

## Role

Cost and token observability.

## External Source

This component is sourced from the actively maintained external repository:
- **Repository**: [AgentSeal/CodeBurn](https://github.com/AgentSeal/CodeBurn)
- **Stars**: ~4.6K
- **Install**: `npm install -g codeburn`

## Why It Exists

Use CodeBurn to find context waste across sessions and projects with an
interactive TUI dashboard.

## Features (External Version)

The external version provides:
- Interactive TUI dashboard with gradient charts
- 13-category task classification (Coding, Debugging, Testing, etc.)
- One-shot success rate tracking
- Multi-provider support (Claude/Codex/Cursor/Pi/Opencode)
- CSV/JSON export
- Native macOS menubar app
- Daily/weekly/monthly cost breakdowns

## Skillkit Mapping

- Install: `./scripts/install-external.sh --skills codeburn`
- Cadence: weekly review or after unusually expensive sessions

## Notes

Observability should reduce tweaking, not become another daily distraction.
The external version at AgentSeal/CodeBurn is actively maintained with
community contributions and regular updates.
