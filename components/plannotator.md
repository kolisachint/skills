# Plannotator

## Role

Visual plan and code review.

## Why It Exists

Use Plannotator-style review when plans or diffs need visual annotation,
approval, or collaborative review before execution.

## Skillkit Mapping

- Skill: `plannotator`
- Fallback: Markdown plan and approval checkpoint.

## Notes

Keep this optional. A visual review loop is helpful for risky work but too heavy
for every small change.
